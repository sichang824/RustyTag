1//! # RustyTag
2//!
3//! `rustytag` 是一个基于 Git 标签的语义化版本管理工具。
4//!
5//! ## 特性
6//!
7//! - 基于 Git 标签的语义化版本管理
8//! - 完整的语义化版本支持
9//! - 自动版本升级（patch/minor/major）
10//! - 本地标签与远程仓库同步
11//!
12//! ## 快速开始
13//!
14//! ```bash
15//! # 初始化语义化版本
16//! rustytag init
17//!
18//! # 升级补丁版本
19//! rustytag patch
20//! ```
21//!
22//! ## 命令详解
23//!
24//! ### 初始化
25//!
26//! ```bash
27//! rustytag init
28//! ```
29//!
30//! 初始化一个新的 Git 仓库并设置语义化版本。
31//!
32//! ### 版本升级
33//!
34//! ```bash
35//! rustytag patch  # 升级补丁版本 (例如: 1.0.0 -> 1.0.1)
36//! rustytag minor  # 升级次要版本 (例如: 1.0.0 -> 1.1.0)
37//! rustytag major  # 升级主要版本 (例如: 1.0.0 -> 2.0.0)
38//! ```
39//!
40//! ### 标签同步
41//!
42//! ```bash
43//! rustytag sync   # 同步本地标签与远程仓库
44//! rustytag reset  # 重置本地标签以匹配远程仓库
45//! ```
